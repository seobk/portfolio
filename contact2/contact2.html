#01 summary

title

금연노트

production period

20.5.13 ~ 20.7.10

type

UI/UX 디자인 프로젝트

role

Teamwork 90%, communication 100%

tools

photoshop, illustrator, adobe XD

흡연하는 사람이 증가한 만큼 금연을 위한 어플을 찾아보게되었고,
‘금연노트’는 불필요한 카테고리와 복잡한 디자인이 사용에 불편함을 주었고, 시각화가 떨어짐

#02 analysis

장점

- 금연의 계기를 부여하고 실제로 사용자 만족도가 높음

- 금연으로 인해 절약된 금액을 한눈에 볼 수 있음

단점

  • - 어플에 강한 아이덴티티가 느껴지지 않고, 불필요한 기능이 많음

    사용량이 적은 기능을 삭제하여 금연시간 레이아웃을 크게 재 조정

  • - 흡연 버튼 클릭 시, 금연 시간 리셋(수정 불가)

    수정 화면 제작

  • - 사명구 영역이 너무 크고, 건강 회복 알림 문구가 눈에 띄지 않음

    사명구 영역을 작게 배치하여 주요 기능을 크게 제작

  • - 통계 수치가 명확하지 않음

    통계 수치 재 정리

#03 improve

  • 카테고리 간소화 및 금연 시간의 가시성

    - 사용량이 적은 기능을 삭제하여 주요 내용인 금연시간이 눈에 띄도록 홈 화면 레이아웃을 재조정

  • 초심자를 위한 가이드 제공

    - 초심자를 위해 사용 방법 제공

  • 메모 기능 추가

    - 기록을 위해 메모 기능 추가

  • 동기 부여 제공

    - 목표 달성 배지 기능을 넣음

  • 통계 카테고리 추가

    - 금연 지속 기간 한 눈에 확인 가능 및 메모, 기록 수정

#04 Objective

쉽고 건강하게 매일 다짐하는 금연노트로, 금연을 목표로 하는 누구나 건강하고 행복한 일상을 누릴 수 있게 도움

#05 Target

보건 복지부, 질병관리본부 등의 자료를 토대로 조사한 결과,
흡연을 하는 사람은 20-40대 남성 / 야근· 육체직 근무자 / 저소득층 / 1인 가구로 집결됨

#06 Persona / UX Flow

나금현(38세)

  • 가족 관계

    - 배우자(30세, 둘째 임신 중), 아들(3세)

  • 사용자 니즈

    - 스트레스를 받지 않고 쉽고 건강하게 금연 하고자 함

  • 숙련 수준/태도

    - 금연에 대한 지식이 전무함

    - 바쁜 업무로 복잡한 어플 사용에 불편함을 느낌

  • 직업

    - 운송업 계약직

  • 사용상 문제점

    - 부가적인 기능들이 많아 한 눈에 확인 하기 어려움

    - 표기가 명확하지 않아 파악하기 어려움

    - 사용 가이드가 없어 어플 사용이 어려움

  • 금연 다짐

    - 흡연으로 건강 악화

  • 어플 탐색

    - 광고 및 복잡한 타 어플

  • 어플 실행

    - 손쉬운 기능,
    주요 기능으로 구성

  • 사용자 설정

    - 금연 각오 작성

    - 금연 시작일 설정

    - 위젯으로 금연 시간 확인

  • 금연 시간 확인

    - 금연 시간을 보며 다짐

  • 금연 성공

    - 건강이 좋아져 일에
    효율이 오름

    - 가족들이 좋아함

우기명(25세)

  • 가족 관계

    - 아버지(58세), 어머니(60세 / 사망)

  • 사용자 니즈

    - 금연으로 인한 절약된 금액을 파악 하고자 함

  • 숙련 수준/태도

    - 주변 환경으로 인해 실천하기 어려움

    - 스마트폰 사용에 능숙함

  • 직업

    - 4년제 대학 복학생
    (편의점 야간 알바, 주말 홍대 버스킹)

  • 사용상 문제점

    - 기록을 할 수 없어서 기능적 흥미를 못느낌

    - 복잡한 메뉴로 혼란을 느낌

    - 한달동안 절약된 금액을 확인 불가

    - 시각적인 재미 요소가 없어 지루함

  • 금연 다짐

    - 담뱃값 지출이 큼

  • 어플 탐색

    - 친구가 어플 소개

  • 어플 실행

    - 핵심 기능만 있어 편리

  • 사용자 설정

    - 절약한 금액 확인을
    위해 담뱃값 기재

    - 금연 시작일 설정

  • 절약 금액 확인

    - 절약한 금액을 보며
    뿌듯함

  • 금연 성공

    - 건강이 좋아짐

    - 절약한 돈으로 버스킹
    새 장비 구매
