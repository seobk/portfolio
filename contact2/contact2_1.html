reward

리워드 아이콘으로 금연 지속 시간 유지 시 획득하는 스폐셜 뱃지 아이콘

카테고리 간소화 및 금연 시간, 절약 금액의 가시성

  • - 메인 화면에 불필요 했던 카테고리를 삭제하고,
    금연 지속 시간과 절약, 소비 금액을 크게 배치하여 한 눈에 파악
  • - 각오(사명구)란을 상단에 배치
  • - 주요 내용이 있는 원형의 레이아웃에 뱃지 모션을 추가
    (금연 지속 시간이 유지될 시, 뱃지 모션이 움직여 재미와 동기 부여 제공)

초심자를 위한 가이드 제공과 메모 기능 추가

  • - 첫 어플리케이션 실행 시 각오(사명구), 담배 가격, 금연 시작일을 작성할 수 있도록 설정 페이지 제공
  • - 메모 기능 추가를 원하시는 사용자들이 많아 하단에 메모 기능 추가

뱃지 기능 추가로 재미 요소와 동기 부여 제공

  • - 하루 금연 성공 시 뱃지 획득 기능을 추가
  • - 획득한 뱃지로 메인 화면에서 뱃지 모션에 움직임을 주어 재미요소와 동기부여 제공

통계 카테고리 추가로 기록을 한 눈에 확인

  • - 금연 중 불가피하게 피운 흡연과 메모 작성한 날을 달력에 원의 형태로 표시
  • - 메모, 흡연 기록 수정 가능
  • - 기간 별 금연 지속 기간 한 눈에 확인 가능

#12 impression

첫 팀원 활동을 하였고, XD와 일러스트를 잘 다루지 못하여 팀원들에게 혹여 피해가 가진 않을까 싶어 더 열심히 하려고 했습니다. 잘 못하는 부분은 팀원들에게 물어 보거나 인터넷을 찾아 보았고, 하나 하나 하다보니 어느덧 능숙하게 할 수 있게 되었습니다.
아이디어를 내고, 좋은 의견이 있을때 적극적으로 얘기 했으며, 회의 할 때도 적극적으로 참여하고 방향을 제시 하였습니다. 팀원들과 함께 좋은 결과물을 만들어 내서 너무 뿌듯했고, 많이 성장 할 수 있었던 좋은 경험이였습니다.
