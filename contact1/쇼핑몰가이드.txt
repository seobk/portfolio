-제작이유: 기존 매직퍼니처 가구 쇼핑몰을 보았을때 너무 복잡해 가독성이 떨어졌고, 내용을 이미지로 처리하여
장애인,노인 등 사용에 어려움이 있는 구매자에겐 쇼핑에 불편함이 있어 구매자가 한정적이다. 이를 보완하여 
전체적으로 눈에 잘 들어오고 어떤 구매자던 쉽게 접근할수 있도록 제작하였다.

-uiux디자인:웹접근성을 고려해 어떠한 사용자던 특별한 기술이 없어도 불편함 없이 쉽게 접근할수
있도록 제작했다.  

-퍼블리싱 가이드:
html: 1. w3c validator 검사에 오류없이 통과한다.
       2. 한문서에 동일한 id를 사용하지 않는다.
       3. 모든 요소와 속성을 소문자를 사용한다.
       4. 들여쓰기를 사용하여 가독성을 높여준다.

css: 1. css 외부문서로 연결해서 사용한다.
      2. 공통된 css는 style.css를 따로 만들어 사용한다.
      3. id,class명은 숫자, 특수문자로 시작할수 없다. 

script: 1. 공통된 script는 외부문서로 연결해서 사용한다.
         2. 끝나는 body 태그 위에 작성한다.
